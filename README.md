# Collecto

> 2D harita tabanlı basit bir oyun, oyuncunun çıkış noktasına ulaşmasını amaçlar.

---

## 🚀 Proje Hakkında

Bir oyuncunun 2D haritada hareket ederek belirli hedeflere ulaşmasını sağlayan bir oyundur. Amaç:

* 🕹️ Oyuncu hareketini yönetmek
* 🌟 Collectible öğeleri toplamak
* 🚪 Harita çıkışına ulaşmak

---

## 🧱 Kullanılan Teknolojiler

* C / C++98
* MinilibX (MLX) grafik kütüphanesi
* 2D grid tabanlı harita
* File I/O

---

## ⚙️ Oyun Mantığı

* Oyuncu, klavye ile hareket eder (W/A/S/D)
* Harita duvarlar, boş alan, collectible ve çıkış noktalarından oluşur
* Tüm collectible toplandığında çıkış aktif hale gelir
* Oyuncu çıkışa ulaştığında oyun biter

---

## 🗺️ Harita Yapısı

* `1` → Duvar
* `0` → Boş alan
* `P` → Başlangıç oyuncusu
* `C` → Collectible
* `E` → Çıkış

## 🧠 Algoritma ve Kontroller

* Klavye input ile oyuncu hareketi
* Harita sınır ve çarpışma kontrolü
* Collectible ve çıkış kontrolü
* Oyun durumu takip ve güncelleme

---

## 🧠 Öğrendiklerim

* 2D oyun mantığı ve grid sistemi
* Klavye input yönetimi
* Grafik kütüphanesi (MLX) kullanımı
* Harita dosyası okuma ve validasyon
* Pointer ve memory management
